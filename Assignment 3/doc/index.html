NfvDeployer RESTful Web Service

The NfvDeployer is a web service based on the NF-FG design.
It allow to:

  • Read all the information about NF-FGs, hosts, catalog and reachable hosts.
  • Deploy a new NF-FG
  • Undeploy an NF-FG
  • Add a single Node in an already deployed NF-FG.
  • Remove a single Node in an already deployed NF-FG.
  • Add a single Link in an already deployed NF-FG.
  • Remove a single Link in an already deployed NF-FG.
